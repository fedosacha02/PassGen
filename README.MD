PassGen is a software like a password manager that creates, stores, organises and shares passwords (within groups, organisations).

The core entities are: 
1. A User/Customer. Depending on the role can also become:
    1) a group administrator;
    2) an organisation administrator;
    3) the organisation superadministrator.
2. A Password belonging to one user completely and to groups/organisations as shared one;
3. A Group consisting of:
    1) user list;
    2) group administrator.
4. An Organisation consisting of:
    1) group list;
    2) organisation administrators;
    3) one organisation superadministrator.
